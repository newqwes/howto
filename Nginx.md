# Nginx

## Установка
```sh
sudo apt update
sudo apt install nginx
```

Файл конфигурации по адресу /etc/nginx/sites-available/default


Проверить файл конфигурации
```sh
sudo nginx -t
```

Проверить файл конфигурации
```sh
sudo service nginx restart
```

Посмотреть статус
```sh
service nginx status
```
